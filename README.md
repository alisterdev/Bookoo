# Bookoo
Academic Project for Web Development course. Fully implemented on-line retail book store.

Technologies used: Java EE, TomEE, JSF 2.0, OmniFaces, RichFaces, PrimeFaces, XHTML, CSS3, Bootstrap, jQuery, MS Project

Key features:
- Purchase and review of books in the store for registered users.
- Efficient managing on the inventory, clients, reviews, banner advertisements, surveys and other content present on the website for managers of the the online store.
- Reporting system for total sales, top sales by various criteria, stock status and more for managers of the online store
